Pokemon

is a small database based on the famous videogame bearing the same name. Pokemon (the DB) contains only one table (also named pokemon) with the following attributes:

- Code: pokedex number of the pokemon
- Name: name of the pokemon
- Type: primary type category of the pokemon (Type 3 fuzzy attribute)
- Group: egg group of the pokemon, a category that determines which pokemon are able to interbreed (Type 3 fuzzy attribute)
- Region: region in which the pokémon was first seen
- Color: perceived color of a pokémon (Type 5 fuzzy attribute) {value/color} where value indicates the possibility of the pokemon color being color
