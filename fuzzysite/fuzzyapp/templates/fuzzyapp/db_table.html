Pokémon

is a small database based on the famous video game bearing the same name (Pokémon). Below is a fraction of the original database displaying selected columns from the pokémon table; elements in this table represent pokémon, a description of the selected attributes follows:

- Code: pokédex number of the pokémon
- Name: name of the pokémon
- Type: primary type category of the pokémon (Type 3 fuzzy attribute)
- Group: egg group of the pokémon, a category that determines which pokémon are able to interbreed (Type 3 fuzzy attribute)
- Region: region in which the pokémon was first seen
- Color: perceived color of a pokémon (Type 5 fuzzy attribute) {value/color} where value indicates the possibility of the pokémon color being color
